# Chronicle Roadmap

Public product roadmap for Chronicle. Status reflects current intent, not guaranteed delivery dates.

---

## ✅ Current

### Chronicle for macOS (R1 – R20)
- Native Tauri + SwiftUI app
- Bill list with due date tracking, amounts, categories
- Local notifications via macOS Notification Center
- Monthly spending overview
- Settings panel (notifications, currency, sort, grouping)
- Widgets (small, medium — Notification Center)
- Menu bar quick-view
- Shortcuts / Siri integration
- iCloud sync (CloudKit)

---

## 🔨 In Progress

### Chronicle for iOS
- Native SwiftUI companion app
- Feature parity with macOS app
- Shared iCloud container
- iOS widgets (small, medium, lock screen)
- Apple Watch app (future R20+)
- Target: R21

---

## 📋 Planned

### Android Companion
- Native Kotlin + Jetpack Compose
- Material Design 3 with Chronicle brand
- Google Drive App Folder sync (or shared container via pairing)
- Google Play Store release
- Notifications, widgets
- Target: R22–R23

### Windows / Linux (PWA + Tauri)
- Progressive Web App built on existing web dashboard
- Tauri wrapper for native Windows binary
- System tray, notifications, auto-start
- Same feature set as macOS app
- Target: R23

### visionOS
- SwiftUI + RealityKit spatial computing app
- Bill list in 3D window arrangement
- Immersive spending chart
- Gesture and voice interactions
- Target: R24

---

## 💡 Future Ideas

These are early-stage ideas with no committed timeline:

### Bank Integration
- Connect via Plaid / TrueLayer to auto-import bills
- Auto-detect recurring charges
- Flag unexpected increases

### Copilot AI Insights
- Natural language bill entry ("Remind me to pay Netflix $15 on the 1st")
- Spending anomaly detection
- Bill negotiation suggestions ("Your internet bill is 20% above average")

### Multi-Currency Support
- Multiple currencies per bill
- Exchange rate tracking
- Conversion in reports

### Collaborative / Shared Bills
- Share a bill tracker with a partner or roommate
- Split-view for shared expenses

### Public API
- Developer API for third-party integrations
- Webhooks for bill events (due today, paid, missed)
